Welcome to the PictoReview prototype

PictoReview is a bundle of prototype tools demonstrating the possibility of creating and exploiting mappings from conference review forms to a generic set of review criteria. The end application showcases the generation of pictorial metaphors of multiple reviews of the same conference paper, leveraging on the mapped values.

TThis first prototype, the mapper, allows you to create a mapping between the fields of a specific conference review form to a set of generic metrics (common for knowledge technology conferences). The mapping can be based on the field names of the original form, or, if you do not want to disclose the field names, you can map them based on their order in the input review form data.

You fill in a form to define a mapping, and the mapper provides you with a JSON header to include in the data you pass to an API that generates the metaphor (see below), and/or with an RDF representation of the mapping with respect to a provisional review metric ontology and the given set of metrics (as a tiny RDF knowledge base).

You can find the mapping functionality here

You can then use the generated header and the set of review data in JSON format to transform your original data to the generic metrics. We provide you with a mock-up of an API call that does this transformation, along with some examples of input data. The output data can be, again, saved as a JSON structure.

The transformation API mock-up can be found here

The third tool is a prototype pictorial metaphor generator. It allows you to upload the JSON file provided by the transformation API mock-up, and outputs a web page with a pictorial scene: each car in the race corresponds to one review. The legend describing the mapping between the review metrics and the pictorial primitives is in the page bottom.

You can try out the metaphor generator here

The research is supported by the VSE IGS project no. 43/2020, „Knowledge Engineering of Researcher Data (KNERD)“ and by the CSF project 18-23964S „Focused Categorization Power of Web Ontologies“ (FCatPoWO).
